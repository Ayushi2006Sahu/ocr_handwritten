Handwritten Text OCR (Streamlit App)
Simple Streamlit app jo handwritten images se text extract karta hai.
EasyOCR → text lines detect karta hai (kahan-kahan likha hai)
TrOCR (Microsoft) → har line ka actual text padhta hai

Sab ek saath install karo:

pip install -r requirements.txt

python -m streamlit run ocr_app.py
